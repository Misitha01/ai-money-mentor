# AI Money Mentor 

An AI-powered financial health analyzer that helps users understand their financial status and get personalized recommendations.

## Features
- Financial Health Score
- AI-based Advice
- Goal-based Planning
- Real-time Analysis

## Tech Stack
- Frontend: React
- Backend: FastAPI
- API: REST

## How it Works
1. User enters income, expenses, savings
2. System calculates financial score
3. AI generates personalized insights

## Impact
- Saves time in financial planning
- Improves savings habits
- Helps achieve financial goals

## Run Project

### Backend
-cd backend
-python -m uvicorn main:app --reload

### Frontend
-cd frontend
-npm start
